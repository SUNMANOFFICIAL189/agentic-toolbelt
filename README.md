# 🏢 CLAUDE HQ — JARVIS Autonomous Agentic System

**The ultimate command center for autonomous development, creative production, and research**

> An autonomous orchestration system powered by the Commander agent. Give it a natural language brief — software, creative, research, operations — and it decomposes, delegates to specialist agents, monitors quality, controls cost, and delivers results.

---

## 🎯 What Is This?

CLAUDE HQ is a **meta-registry + orchestration brain** for Claude Code. It contains:

1. **The Commander** → `commander/COMMANDER.md` — Orchestration agent that decomposes any task, delegates to specialists, enforces quality gates and cost control
2. **Tool Registry** → `registry.json` — Catalog of 25 tools, 256+ skills, 138 subagents
3. **Operating Principles** → Boris Cherny's engineering philosophy, zero-cost-first protocol, credential safety
4. **Predefined Stacks** → 11 tool combinations for common workflows

## 🎯 Core Principles

- **Zero Cost First** — Free alternatives before spending. Any cost requires approval.
- **Cheapest Viable Model** — Haiku → Sonnet → Opus based on complexity.
- **Plan Before Execute** — Mission board before agents spawn.
- **Staff Engineer Bar** — Quality gate on every deliverable.
- **Learn From Mistakes** — Every correction → rule in LESSONS.md.
- **Never Expose Credentials** — `.env` files only, never in git or logs.
